File Structure

Path Description
documentation The documentation for the theme
demo All the files for theme demo
assets Images and favicon
components Templates for the HTML components used throughout the theme
js Javascript files for components that require it
styles Theme styles
css Contains main.css file which has all of the theme's styles
sass Contains .scss files which has been compiled to main.css
abstracts Mixins for media queries
base CSS reset, color variables, and typography styles
components Styles for HTML components
utilities Utility classes for animations, spacing, and typography
index.html Homepage
